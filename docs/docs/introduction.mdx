## How it works

**HTML To Gutenberg** uses an Abstract Syntax Tree (AST) to parse your HTML block files, preserving your original markup structure. It interprets custom attributes and transforms this tree into all the necessary Gutenberg block assets â€” including metadata, editor components, frontend renderers, and block registration files.
